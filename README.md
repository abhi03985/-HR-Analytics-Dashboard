# 📌 Project Overview

An interactive **HR Analytics Dashboard** built using **Power BI, SQL, Excel, and DAX** to analyze employee attrition, workforce demographics, job satisfaction, department performance, and HR KPIs. The dashboard enables HR professionals to make data-driven decisions through interactive visualizations and filters.

---

## 📊 Dashboard KPIs

| KPI | Value |
|------|------:|
| 👨‍💼 Total Employees | **1,470** |
| ✅ Active Employees | **1,233** |
| ❌ Attrition Count | **237** |
| 📉 Attrition Rate | **16.1%** |
| 🎂 Average Age | **37 Years** |

---

## 🚀 Dashboard Features

✅ Employee Attrition Analysis

✅ Department-wise Performance

✅ Job Satisfaction Ratings

✅ Monthly Income Analysis

✅ Education Field Distribution

✅ Age Band Analysis

✅ Interactive Filters & Slicers

✅ Dynamic KPI Cards

---

## 💡 Key Insights

📌 Sales department experienced the highest employee attrition.

📌 Research & Development has the largest workforce.

📌 Employees aged **25–34 years** form the biggest employee segment.

📌 Research Scientists show comparatively higher job satisfaction.

📌 The dashboard helps HR teams identify workforce trends and improve retention strategies.
